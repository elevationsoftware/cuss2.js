Component(component, cuss2, _type)

General object representing a CUSS component with methods and properties to interact with it.

# new Component(component, cuss2, _type)

Parameters:
Name Type Description

cuss2 Cuss2

Properties:
Name Type Description
id number Numeric ID assigned to the component; used for identification of a specific component.
onmessage Subject.<PlatformData> Observable that emits when a message is received from the component.
required boolean Whether the component is required to be connected to the CUSS Platform.
statusChanged BehaviorSubject.<StatusCodes> Observable that emits the status of the component on changes.
deviceType DeviceType The type of device the component is, *See IATA documentation for more details.
pendingCalls number The number of pending calls to the component.
enabled boolean Whether the component is enabled or not.
pollingInterval number The interval in milliseconds to poll the component for data.
parent any The parent of the component.
subcomponents Array.<Component> The subcomponents of the component.
readyStateChanged BehaviorSubject.<boolean> emits true when the component is ready

this.enabled
